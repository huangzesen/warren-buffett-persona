## 示例格式

每个示例文件包含：
1. **基本信息** — 年份、主题、URL
2. **精选段落** — 原文 + 声音标注
3. **微层面分析** — 用词、节奏、破折号等
4. **综合分析** — 这一年的声音特征
5. **写作技巧总结** — 可复制的技巧
6. **如何临摹** — 模板和检查清单
7. **经典引用库** — 核心引语
